Com o plugin devidamente instalado no seu navegador, clique em Selenium IDE.

![alt text: Apresentado na imagem o navegador com a janela suspensa exibindo os plugins instalados no Chrome e a seleção do Selenium IDE.](https://cdn3.gnarususercontent.com.br/2377-donet-testes-curso3/01/aula1-img10.png)


Informe a URL alvo para a criação para a gravação do passo a passo.
  

![alt text: A imagem mostra a ferramenta Selenium IDE evidenciando o local de inserção da URL e o botão “Rec” de gravação.](https://cdn3.gnarususercontent.com.br/2377-donet-testes-curso3/01/aula1-img11.png)


Depois vamos clicar em “RUN” e executar o script gerado quantas vezes for necessário para uma bateria de testes.
  

![alt text: A imagem mostra a ferramenta Selenium IDE com um script de automação gerado na tela principal e é evidenciado o botão de “Play” para a execução do script.](https://cdn3.gnarususercontent.com.br/2377-donet-testes-curso3/01/aula1-img12.png)

Para exportar o script para um código C#, Java ou Python clique com o botão direito sobre o script de teste salvo no lado esquerdo da IDE e clique em exportar.
  

![alt text: A imagem mostra a ferramenta Selenium IDE com uma janela suspensa com as opções de linguagem de programação para exportação do script gerado.](https://cdn3.gnarususercontent.com.br/2377-donet-testes-curso3/01/aula1-img13.png)
Pronto, feito isso teremos um novo arquivo com extensão “.cs” com o script do passo a passo em um código em linguagem de programação que podemos aproveitar na escrita de futuros testes de interface.
